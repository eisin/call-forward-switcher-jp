転送電話サービス転送先切り替え

call-forward-switcher-dcmは、転送電話サービスにおける転送先を変更するものです。

特に、複数人で携帯電話を受け付ける窓口業務において、日時を予め定めておくことによってその受付先を変更することを意図しています。

概要

call-forward-switcher-dcmの概要を示します

転送元電話番号の前提事項

転送元電話番号は、日本国内におけるNTTドコモ社のサービスによるものとします。 転送元電話番号において、転送電話サービスを契約しており、かつ遠隔操作を有効にしているものとします。

転送先の変更方法

NTTドコモ社では転送電話サービスの設定変更を、遠隔操作により行うことができます。 遠隔操作による設定変更は本来人間がダイヤル操作することを想定していると思われますが、 これを機械的にダイヤル操作することによって実現します。

ダイヤル操作はTwilioによって行います。人間は操作に介入しません。

転送先変更の確認方法

転送電話サービスの転送先変更が確実に行われたかどうか、応答音声を認識させることによって行います。 音声認識の精度は100％とは言えないものの、少なくとも転送先変更が行われていないことが確実な場合には検知できます。

音声認識は、Google Cloud Voice APIを利用することによって行います。
